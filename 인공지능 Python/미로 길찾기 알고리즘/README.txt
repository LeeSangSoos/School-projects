파일은 core_code.zip을 열어서 원래 pymaze-master폴더에 src폴더 기준으로 위치에 맞게 그대로 열어서 덮어쓰면 됩니다.

Uniform cost search실행:
UCS2.py 라는 파일을 실행하면 됩니다

A* search실행:
A_search.py 라는 파일을 실행하면 됩니다

제 환경에서는 mp4저장은 실패했어서 영상 파일을 저장하려면 따로 찍어야 했습니다.


every_code 파일은 제가 작업한 가상환경 폴더까지 포함한 총 파일입니다. 혹시라도 core_code.zip에 코드만으로 작동이 안될 때를 걱정했습니다.
